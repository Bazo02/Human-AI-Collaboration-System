Plots

{% if results.plots.accuracy %}

Accuracy

How it is calculated: Accuracy is first computed per participant and condition as the proportion of correct decisions. The plotted value is the mean of those participant-level accuracy values for each condition.

{% endif %} {% if results.plots.time %}

Decision Time

How it is calculated: Decision time is first computed per participant and condition as the mean time in seconds. The plotted value is the mean of those participant-level time values for each condition.

{% endif %} {% if results.plots.trust %}

Trust

How it is calculated: Trust is computed from the trust questionnaire items (trust_q1, trust_q2, trust_q3). First, an average trust score is calculated per participant. The plot shows the mean of those participant-level trust scores.

{% endif %} {% if results.plots.sus %}

SUS

How it is calculated: SUS (System Usability Scale) is scored using the standard method: for odd-numbered questions, score = response − 1; for even-numbered questions, score = 5 − response. The adjusted scores are summed and multiplied by 2.5 to produce a 0–100 SUS score. The plot shows the mean participant-level SUS score.

{% endif %} {% if results.plots.ai_followed %}

AI Followed Rate

How it is calculated: AI followed rate is computed in the AI condition only. For each participant, it is the proportion of AI-condition cases where the participant’s decision matched the AI recommendation. The plotted value is the mean of those participant-level rates.

{% endif %} {% if results.plots.ai_confidence %}

AI Confidence

How it is calculated: AI confidence is averaged across AI-condition decisions for each participant. The plotted value is the mean of those participant-level average confidence values.

{% endif %} {% if results.plots.ai_prob_approve %}

AI Approval Probability

How it is calculated: The model’s approval probability is averaged across AI-condition decisions for each participant. The plotted value is the mean of those participant-level average probabilities.

{% endif %} {% if results.plots.explanation_open_rate %}

Explanation Open Rate

How it is calculated: Explanation open rate is computed per participant and condition as the proportion of cases where the explanation was opened. The plotted value is the mean of those participant-level rates.

{% endif %} {% if results.plots.ai_seen_rate %}

AI Seen Rate

How it is calculated: AI seen rate is computed per participant and condition as the proportion of cases where the AI panel was present. The plotted value is the mean of those participant-level rates.
